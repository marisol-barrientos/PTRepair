# PTResolver

This repository contains the research prototype implementation of **PTResolver (Process Tree Resolver)** for the paper *Optimizing Compliance Violation Resolution in Business Process Models*.

PTResolver is a framework for generating, applying, and assessing resolution strategies for compliance violations in business process models.

PTResolver combines:
- Large Language Model-based generation of candidate resolution strategies
- Rule-based process model transformations
- Assessment of generated strategies and resulting process models

PTResolver supports multiple compliance perspectives, including: Control-flow, data, resource, and temporal aspects.

This repository is currently being reorganized and updated.  
Documentation, navigation, setup instructions, and examples are still being prepared.
